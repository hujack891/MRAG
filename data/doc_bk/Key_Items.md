# Key Items

Key Items are an integral part of the progression in Black Myth: Wukong"). Some of them are placed naturally along the main path through the game. However, quite a few are hidden in secret areas or locked behind optional bosses. These key items can open up new secret areas, spawn new bosses, and, in some cases, allow you to get new loot, upgrades, and resources. 

In this guide, we will go over each of the Key Items and where you can find them. You can also find details on what each of these Key Items does and where to use them.
